Example Bootstrap Carousel C

Another bootstrap carousel example. In this example I add more images. In example B I added more images to each of three items, so each time was a vertical stack of nine images, for a total of 27 images. In this example, I add two additional stacks of nine images to each item. Now each of the three items has 27 images for a total of 81 images, but only three carousel items. An efficient way to scroll through many images quickly.
